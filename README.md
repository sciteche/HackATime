# First Aid Guide

A simple website providing essential first aid information for common emergencies.

## Categories Covered

- Burns
- Cuts and Wounds
- Choking
- Heart Attack
- Fractures
- Poisoning
- Allergic Reactions

## Disclaimer

This is not a substitute for professional medical advice. Always consult healthcare professionals for serious injuries.

## How to Use

Open `index.html` in your web browser to view the guide.
